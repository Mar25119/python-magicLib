# Техническое задание

## 1. Описание задачи
Разработать утилиту, которая принимает путь к файлу и выводит:
- MIME-тип (например, `image/png`)
- Человекочитаемое описание (например, `PNG image data, 1920 x 1080`)

Утилита должна использовать библиотеку `python-magic`.

## 2. Цель работы
Освоить базовое применение `python-magic` для безопасного определения типа файла по его содержимому.

## 3. Функциональные требования
- Реализовать функцию `detect_file_type(filepath: str) -> dict`, возвращающую словарь с ключами `"mime"` и `"description"`.
- Обрабатывать несуществующие или недоступные файлы корректно (без аварийного завершения).
- Поддерживать любые файлы: изображения, документы, архивы и т.д.

## 4. Нефункциональные требования
- Код должен быть читаемым и легко тестируемым.
- Использовать только `python-magic` (без `mimetypes` или проверки расширений).
- Обработка ошибок через `try-except`.

## 5. Ограничения и допущения
- Предполагается, что `libmagic` установлен в системе.
- Не требуется поддержка потоков (только файлы на диске).
- Библиотека `python-magic` установлена (`pip install python-magic`).

## 6. Источник ТЗ
Самостоятельная постановка.
